# Discover Corralejo — Design Phase 1

This update applies the approved visual direction across the reusable PWA interface.

## Visual system

- Deep Atlantic Teal: `#155D6C`
- Warm Sand: `#F4EBDD`
- Warm White: `#FFFDF8`
- Coral / Sunset: `#FF6C66`
- Muted Ocean Blue: `#72AFC1`
- Charcoal: `#252525`
- Warm Ochre: `#D6A44A`

## Updated screens

- Immersive welcome and eight-language selection screen.
- Editorial route overview with progress and continue-tour controls.
- Category-aware route cards for Main Stops, Secret Photo Spots and Hidden Gems.
- Walking-friendly stop pages with a prominent audio card, transcript drawer, horizontal media gallery and sticky completion control.
- Approved in-app walking map with a compact completed-stop summary and next-stop actions.

The design is data-driven, so the same components automatically apply to all 18 stops and can be reused for future tours.
