# Discover Corralejo — Real In-app Map

The navigation screen uses the local `assets/maps/corralejo.pmtiles` basemap, MapLibre GL JS and saved GeoJSON walking routes.

## Current coverage

- Real Corralejo streets, buildings, coastline and paths.
- All 17 approved stop-to-stop walking routes.
- Current-stop and next-stop markers.
- Walking distance and estimated time.
- Offline map and route support after the tour download is completed.
- Google Maps fallback for optional live directions.

The interface follows the Atlantic editorial visual system introduced in Design Phase 1.
